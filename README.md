Low-level web server written in Swift.

Implements a web server from scratch using raw TCP sockets, without relying on
higher-level frameworks. Handles HTTP request parsing, routing, and response
generation manually.

Features:
- Custom HTTP parser for request handling
- Router system for path-based routing
- Concurrent connection handling using dispatch queues
- Static resource serving
- Dynamic page generation

Built to power a personal website and demonstrate low-level networking concepts
in Swift.
